TreeCapitator Compatibility

TreeCapitator is designed to work seamlessly with various Minecraft Add-Ons. Below is a list of known compatible Add-Ons.

Add-On Name Creator Remarks
Better On Bedrock Poggy Full compatibility with all tree types and custom wood blocks.
RealismCraft Spark Universe Full compatibility. TreeCapitator adapts to RealismCraft's unique tree structures.
Tinker's Construct FTB Studios Compatible with Tinker's Construct axes. No axe customization for Tinker's axes.
Forest Craft Float Studios Full compatibility. Works with all custom trees and the custom Shear Axe.
Biomes Add-On Block Factory Compatible with all custom biome trees and vegetation.
Biomes + Structures Add-On Mythicus Compatible with all custom biome trees and vegetation.
Farming Podcrash Compatible with custom trees.
More Ores & Tools Netherpixel Full support for all custom axes.
Nature Builders Horizon TreeCapitator works with custom trees, but some blocks will be left floating.
Note: Compatibility is based on the latest versions of both TreeCapitator and the listed Add-Ons. Always ensure you're using the most recent updates for the best experience.
Note: Is your Add-On compatible but not on the list? Let me know by contacting me, and I'll be sure to add it here :)
