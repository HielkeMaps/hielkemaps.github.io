Added
  • Support for RealismCraft pale oak trees
Added
  • Support for pale oak trees
Fixed

  • Add-On creators can now add compatibility for TreeCapitator by themselves!

Changed

Fixed

Added

  • Support for Forest Craft Add-On
  • Support for Farming Add-On
Improved
  • Leaf detection algorithm for better performance
Fixed
  • Compatibility issues with some custom add-ons
Added
  • Support for Tinker's Construct Add-On
  • New configuration options for custom add-on integration
Fixed
  • Issue where cherry leaves were not dropping loot when using fast decay
Added
  • Fast Decay leaf break option
  • Support for RealismCraft Add-On
  • Compatibility with azalea trees
Fixed
  • Issue where cherry trees sometimes didn't work properly
Improved
  • Tree detection algorithm for better accuracy
  • Initial release of TreeCapitator
  • Basic functionality for cutting down entire trees
  • Support for vanilla Minecraft trees
  • Configurable settings for activation method and tool usage
